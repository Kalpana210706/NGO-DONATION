#  NGO Donation System

A full-stack web application that enables users to securely donate to NGOs using an online payment gateway.  
The system supports user authentication, donation tracking, and an admin panel for monitoring donations.

##  Features

###  User Features
- User Registration & Login (JWT Authentication)
- Secure Dashboard
- Make Donations (PayHere Sandbox Integration)
- View Donation History (Pending / Completed)
- Logout functionality

###  Admin Features
- View all donations
- Monitor donation status
- Secure admin-only routes

##  Tech Stack

## Frontend
- React.js
- Axios
- Tailwind CSS
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- PayHere Payment Gateway (Sandbox)

### Deployment
- Backend deployed on **Render**
- Frontend can be deployed on **Netlify / Vercel**
